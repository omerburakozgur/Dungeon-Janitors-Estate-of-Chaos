Features

Input
	Change the default input system that I've given to your system or modify it.

Interaction
	Change the interaction system that I've given to your system or modify it.
	
Stains
	Customizing the stain behaviours from scripts.
	Customizing the stain visuals.
	Customizing the stain data from sciptable object data.
		Changing the tools required to clean from scriptable object.
		Changing the decal materials from the scriptable object data.
		Changing the HP, regen rate etc.


Cleaning Tools
	Changing the effective stains of the tools.
	Adding or changing the cooldown value after the tool is used.
	Customizing the tool data from sciptable object data.
		Change the damage that the tool deals to the stains.
		Change the range of the tool.
